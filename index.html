Notre projet

 Le site météo est une application web qui permet aux utilisateurs de vérifier la température et les conditions météorologiques. Il est construit en utilisant HTML, CSS, JavaScript et une API météo. Le front-end est conçu en utilisant HTML pour la structure, CSS pour le style et JavaScript pour gérer l'interactivité et la récupération de données. Le site affiche bien en évidence la température et fournit une brève description des conditions météorologiques, telles que ensoleillé, nuageux ou pluvieux. Il est entièrement réactif, permettant l'accès depuis n'importe quel appareil, et intègre la géolocalisation pour fournir des informations météorologiques locales précises.
